Texto em Negrito / Destaque

Termos em Negrito usando a tag <b> são ultilizados para dar forma e não semantica ao termo.
Para dar destaque e significado ao termo é usado a tag <strong>.

Texto em Itálico / Ênfase

Termos em Itálico com forma e sem semantica usamos a tag <i>.
já a tag <Em> dar ênfase ao termo.

Texto de Marcação

Podemos criar um texto marcado usando a tag <mark>, sua cor padrão é amarelo.
A cor pode ser modificada na HTML mais o certo é modificar usando as CSS.
